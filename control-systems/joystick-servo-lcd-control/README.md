# Joystick Servo LCD Control

Proyecto de control de posición angular con Arduino UNO, joystick analógico, servomotor SG90 y pantalla LCD 16x2.

## Objetivo

Usar el eje X del joystick como referencia de posición para el servo y mostrar el estado del sistema en una pantalla LCD.


## Estado actual

Versión funcional completada:
- lectura analógica del joystick
- conversión a consigna angular
- control de posición del servo SG90

Parte aplazada:
- integración de LCD 16x2 en modo 4 bits

Motivo:
- necesidad de mejorar la calidad de soldadura y la fiabilidad del conexionado del módulo LCD antes de continuar la integración
